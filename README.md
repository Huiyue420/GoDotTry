# 簡易多人連線對戰遊戲 (Godot 4)

這是一個使用 **Godot Engine 4.x** 開發的簡易多人線上射擊遊戲原型 (Prototype)。展示了 Godot 內建的高階多人連線 API 功能。

## 🎮 遊戲功能

1.  **多人連線大廳**:
    *   **建立房間 (Host)**: 啟動伺服器並生成玩家。
    *   **加入房間 (Join)**: 輸入 IP (預設 localhost) 加入現有遊戲。
2.  **角色操作**:
    *   **移動**: 上下左右 (WASD 或 方向鍵)。
    *   **戰鬥**: 滑鼠跟隨瞄準，左鍵射擊。
3.  **同步機制**:
    *   使用 `MultiplayerSynchronizer` 同步玩家位置與旋轉。
    *   使用 `MultiplayerSpawner` 動態生成玩家與子彈。
4.  **遊戲規則**:
    *   每位玩家初始血量為 **3** 滴。
    *   被子彈擊中扣 1 滴血。
    *   血量歸零時角色死亡 (消失)。

## 🛠️ 開發環境

*   **Godot Version**: 4.5 (或是 4.0 以上版本)
*   **Language**: GDScript

## 🚀 如何執行與測試

由於這是多人連線遊戲，需要在單機上模擬多個客戶端來測試：

1.  **下載專案**:
    ```bash
    git clone https://github.com/Huiyue420/GoDotTry.git
    ```
2.  **匯入 Godot**:
    *   開啟 Godot 引擎。
    *   點擊 `Import` 並選擇專案資料夾中的 `project.godot`。
3.  **設定多開測試**:
    *   在 Godot 編輯器上方選單，點擊 **Debug (除錯)** -> **Customize Run Instances... (自訂執行實體)**。
    *   勾選 **Enable Multiple Instances (啟用多個實體)**。
    *   將實體數量設為 **2** (或是更多)。
4.  **開始遊戲**:
    *   按下 `F5` 或點擊右上角的 Play 按鈕。
    *   系統會開啟兩個遊戲視窗。
    *   **視窗 A**: 點擊 `Host` 當作主機。
    *   **視窗 B**: 保持 IP 為 `127.0.0.1`，點擊 `Join` 加入遊戲。
    *   現在你可以看到兩個玩家並進行對戰了！

## 🕹️ 操作方式

*   **移動**: `W`, `A`, `S`, `D` 或 `↑`, `↓`, `←`, `→`
*   **瞄準**: 滑鼠游標位置
*   **射擊**: 滑鼠左鍵

## 📂 專案結構

*   `main.tscn`: 遊戲入口、UI 介面與網路生成節點。
*   `player.tscn`: 玩家場景 (含同步設定)。
*   `bullet.tscn`: 子彈場景。
*   `main.gd`: 處理連線邏輯 (Host/Join) 與玩家生成。
*   `player.gd`: 處理移動輸入與 RPC (遠端呼叫) 射擊請求。
